# OppaDrama CloudStream Extension

This is the official CloudStream extension repository for OppaDrama.

## Installation

Add this repository in CloudStream by pasting this URL in the **Repository URL** field:

`https://raw.githubusercontent.com/lutpong/cloudstream-extensions/builds/repo.json`

## Features

* OppaDrama Provider
* Indonesian Subtitles
* Fast scraping
